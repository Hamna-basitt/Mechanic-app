# Mechanic Registration App (Flutter)

A Flutter application for mechanics to register themselves, upload profile picture, CNIC (front & back), and provide location & professional information. Designed with a user-friendly interface for easy registration and dashboard access.

---

## Features

- Multi-step registration form:
  - Personal Information (Name, Phone, Password, Profile Picture)
  - Location Information (Shop & Home Address)
  - Professional Information (Mechanic Type, Experience, Working Hours)
  - Documents (CNIC Front & Back)
- Upload and preview images for profile and CNIC.
- English and Urdu labels for better user understanding.
- Progress indicator and step navigation.
- Secure password input with visibility toggle.
- Responsive design compatible with Android & iOS.
